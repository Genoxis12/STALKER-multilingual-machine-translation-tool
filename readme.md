# __ENGLISH__

## INFORMATIONS

This is an automatic translation tool designed for mods of the S.T.A.L.K.E.R franchise.
</br>

The translation is not and never will be perfect; it is carried out by a bot and therefore contains errors. What's more, only xml files are translated.
</br>

There are plans to add features to perfect the translation and translate other files.
<br/>

Please let me know of any problems.
<br/>

## INSTALLATION GUIDE

1.
    - Download the project as a zip file
    - or make a git clone

2. Create a DeepL API key and add it to the common.api file:
    > API_KEY = "xxxxxxxx-xxx-xxxx-xxxx-xxxxxxxxxxxx"

    (https://www.deepl.com/en/your-account/keys)
3. Run the multilingual-machine-translation-tool.bat file

## HOW THE SCRIPT WORKS

When you enter the path of the folder you want to translate, the script will recreate this folder by translating the XML files. If a `localization.ltx` file exists, the XML files will be automatically reorganized.

## PROBLEMS?

**API problems**
if you encounter these errors: 
> No module named 'common.api'

>cannot import name 'API_KEY' from 'common.api'

You need to create an api file in the common folder and add this line: 
> API_KEY = "xxxxxxxx-xxx-xxxx-xxxx-xxxxxxxxxxxx"

change the code with your API key (https://www.deepl.com/en/your-account/keys).
DeepL offers free API keys with a limit of 500,000 characters / month.

**Other**
If you have any problems, please feel free to send me a private message.
